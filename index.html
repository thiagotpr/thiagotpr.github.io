Javascript 1995 Interpretação Java, Python e Perl Desenvolvimento web;
Criação de aplicativos web;
Desenvolvimento de jogos web;
Desenvolvimento de aplicativos móveis híbridos. 		Muitos iniciantes começam a aprender programação através dele. No entanto, a complexidade pode aumentar à medida que você explora tópicos avançados e o comportamento da linguagem no computador. Multiparadigma (orientado a objetos, imperativo, funcional). JavaScript é amplamente considerada a linguagem de programação mais popular em 2023, principalmente devido ao seu papel central no desenvolvimento web e na criação de aplicativos web interativos. É uma linguagem essencial para o desenvolvimento web, permitindo a criação de interações dinâmicas em páginas da web. Achei ela bastante legal e um pouco desafiadora, mas não vejo a hora de me aprofundar mais nela.
Java 1996 Interpretação e Compilação C++ Desenvolvimento de aplicativos Android; Desenvolvimento de aplicativos empresariais e sistemas de backend; Desenvolvimento de jogos; Aplicações científicas e financeiras. Sua sintaxe é mais rigorosa do que Python, mas muitas ferramentas de desenvolvimento e recursos de aprendizado estão disponíveis. É uma linguagem tipada estaticamente, o que pode exigir uma compreensão sólida de tipos. Orientada a objetos. continua sendo uma linguagem amplamente usada, especialmente em aplicativos empresariais, desenvolvimento Android e em sistemas críticos. Embora tenha perdido um pouco de popularidade, ainda é uma das linguagens mais usadas no mundo. Conhecida por sua portabilidade, segurança e estabilidade. Sua sintaxe é mais rigorosa em comparação com algumas outras linguagens, creio que seja um pouco complicado de aprender, ainda não me aprofundei tanto nela.
PHP 1995 Interpretação C e Perl Desenvolvimento web do lado do servidor; Construção de sites dinâmicos e aplicações web; Aplicações de comércio eletrônico; Aplicações web baseadas em formulários e bancos de dados. Frequentemente considerada uma das linguagens mais fáceis de aprender para iniciantes. Sua sintaxe é simples e legível, o que facilita a compreensão do código. Python também possui uma grande comunidade de suporte e uma ampla gama de recursos de aprendizado, incluindo tutoriais e cursos online. Multiparadigma (orientado a objetos, imperativo) PHP é uma linguagem que é amplamente usada no desenvolvimento web do lado do servidor, mas perdeu parte de sua popularidade em comparação com os anos anteriores. É fácil de aprender para tarefas web simples, boas práticas de segurança são essenciais para evitar vulnerabilidades.
Python 1990 Interpretação ABC e Perl Desenvolvimento web (Django, Flask). Ciência de dados e análise de dados; Automação de tarefas e scripts; Automação de testes. Sua curva de aprendizado é geralmente mais suave para iniciantes, pois é focada principalmente em tarefas web. No entanto, boas práticas de segurança podem ser necessárias para evitar vulnerabilidades. Multiparadigma (orientado a objetos, imperativo) Python teve um crescimento notável em popularidade nos últimos anos, impulsionado pelo seu uso em ciência de dados, aprendizado de máquina, automação e desenvolvimento web. É amplamente elogiada por sua simplicidade, legibilidade e versatilidade. É frequentemente recomendada para iniciantes devido à sua sintaxe clara e à grande comunidade de suporte. Já tive monitoria de Python na escola e achei incrível.
C# 2002 Interpretação e Compilação Java e C++ Desenvolvimento de jogos; Desenvolvimento de aplicativos Windows; Desenvolvimento de aplicativos móveis; Desenvolvimento de aplicativos de desktop e empresariais. C# é uma linguagem tipada estaticamente e é frequentemente usada no desenvolvimento de jogos e aplicativos Windows. Pode ser um pouco mais desafiador para iniciantes devido à complexidade do ambiente de desenvolvimento. Orientado a objetos. C# é popular no desenvolvimento de jogos e aplicativos Windows, mantendo uma base de usuários sólida, mas mais específica em relação a certos ecossistemas. C# é uma escolha popular para o desenvolvimento de jogos com a Unity e para aplicativos Windows. Foi a primeira linguagem de programação no qual eu tive contato, com ela, fiz dois jogos no Unity na época do ensino médio.
Kotlin 2011 Interpretação e Compilação Java, C# e Groovy Desenvolvimento de aplicativos Android; Desenvolvimento de backend; Desenvolvimento web; Aplicativos desktop. Kotlin é considerada relativamente fácil de aprender, especialmente para desenvolvedores Java, devido à sua interoperabilidade com Java. A sintaxe é mais concisa e expressiva em comparação com Java, tornando-a mais amigável para iniciantes. Multiparadigma (orientado a objetos, funcional). A popularidade do Kotlin tem aumentado significativamente desde que se tornou a linguagem oficial para desenvolvimento Android. Em rankings de popularidade de linguagens de programação, Kotlin frequentemente ocupa uma posição elevada. É elogiada por sua concisão e recursos de segurança. Sua interoperabilidade com Java é uma vantagem significativa. Não sei muito sobre ela, mas agora vou aprender para poder usar na residência.
